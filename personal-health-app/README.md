# VegFit Planner

A browser-based vegetarian health and gym planner.

## Features

- Profile-based protein and calorie estimates
- 12-hour meal timing around gym schedule
- Strict vegetarian or vegetarian + eggs mode
- Daily gym log with muscles, effort, sleep, soreness, and notes
- Weekly muscle coverage tracking
- Rest and recovery recommendations
- Grocery list
- Movable dashboard sections saved locally
- Phone sharing through an encoded plan link
- Installable/offline-friendly browser app files

## Run locally

Open `index.html` in a browser.

## Phone usage

Use **Share to phone** after updating the plan. Opening the shared link on another device imports that plan into that device's browser storage.

Automatic cross-device syncing needs a backend or database. This version keeps data private in each browser unless you share/import it.
